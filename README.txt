Relational DataModel

Collaborators: Shoham Shitrit and Miller Hickman
NETID's: sshitrit, mhickman

To build: gcc -std=c99 -Wall -Werror database.c LinkedList.c CP.c CR.c CDH.c CSG.c SNAP.c -o run

To run: Use executable file ./run. First, all the Relations will print out, and then the requested examples from part one will be shown. Then, for the queries, the user will be prompted to enter fields for it. Then, examples from part three will be shown. Finally, the user will be asked if he/she wants to read and write the file.
